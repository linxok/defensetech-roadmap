## asyncio для телеметрії

asyncio дозволяє обробляти тисячі мережевих з’єднань без блокування.

```python
import asyncio

async def read_telemetry():
    reader, writer = await asyncio.open_connection('127.0.0.1', 14550)
    while True:
        data = await reader.read(1024)
        if not data:
            break
        print(data.hex())

asyncio.run(read_telemetry())
```

## FastAPI + WebSocket

```python
from fastapi import FastAPI, WebSocket

app = FastAPI()

@app.websocket("/ws")
async def websocket_endpoint(websocket: WebSocket):
    await websocket.accept()
    while True:
        data = await websocket.receive_text()
        await websocket.send_text(f"Message text was: {data}")
```
