# Домашнє завдання 09

## Завдання

ROS2: створити ROS2 міст для телеметрії

## Опис

Виконайте розширену версію практичного завдання та опублікуйте результат.

## Що зробити

1. Прочитайте офіційну документацію з `resources.md`.
2. Реалізуйте розширену версію практичного завдання.
3. Додайте unit-тести або integration-тести.
4. Напишіть README з інструкцією запуску.
5. Запишіть нотатку: що вийшло, що складно, наступні кроки.

## Критерії оцінювання

- Код компілюється/запускається.
- Є документація та інструкція.
- Додано обробку помилок.
- Покрито тестами або чеклістом.
