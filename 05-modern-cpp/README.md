# 05. Сучасний C++

## C++ у робототехніці

C++ залишається мовою для performance-critical компонентів: flight controllers, vision pipelines, DDS, embedded SDK. Сучасний C++ (C++17/20) дає зручні інструменти без втрати швидкості.

## Теми модуля

- **STL** — vector, map, unordered_map, algorithm.
- **Smart pointers** — unique_ptr, shared_ptr, weak_ptr.
- **Move semantics** — оптимізація передачі об’єктів.
- **Multithreading** — thread, mutex, condition_variable, future.
- **Atomics** — lock-free синхронізація.
- **CMake** — збірка кросплатформених проєктів.

## Thread-safe черга повідомлень

```cpp
#include <queue>
#include <mutex>
#include <condition_variable>

template <typename T>
class MessageQueue {
    std::queue<T> q_;
    std::mutex m_;
    std::condition_variable cv_;
public:
    void push(T value) {
        std::lock_guard<std::mutex> lock(m_);
        q_.push(std::move(value));
        cv_.notify_one();
    }
    T pop() {
        std::unique_lock<std::mutex> lock(m_);
        cv_.wait(lock, [this] { return !q_.empty(); });
        T value = std::move(q_.front());
        q_.pop();
        return value;
    }
};
```
