# Апаратні рекомендації

## Для навчання

- ПК з 16+ GB RAM, SSD, багатоядерний CPU.
- Достатньо для SITL, Docker, ROS2, CV-моделей.

## Embedded / companion computer

- **Raspberry Pi 4 / 5** — бюджетний companion computer.
- **NVIDIA Jetson Nano / Orin** — для AI і CV на edge.
- **Orange Pi / Radxa** — альтернативи.

## Flight controllers

- **Pixhawk 6C / 6X** — для ArduPilot / PX4.
- **Cube Orange** — професійний.
- **Matek H743** — бюджетний.

## Телеметрія

- **SiK Radio** (433 / 915 MHz).
- **RFD900** — дальній зв’язок.
- **Herelink** — HD video + control.

## Камери

- **Raspberry Pi Camera Module** — для CV.
- **USB UVC camera** — універсальна.
- **FLIR** — тепловізор.

## Безпілотник для тестів

- FPV-коптер на Betaflight для ручних навичок.
- Симулятор (SITL) — основний інструмент для розробки.
