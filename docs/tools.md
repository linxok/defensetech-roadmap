# Рекомендовані інструменти

## Розробка

- **VS Code** з розширеннями: Python, C++, ROS, Docker, Markdown, Mermaid.
- **PyCharm** — для Python-проєктів.
- **Clion** — для C++.

## Мережа

- **Wireshark** — аналіз трафіку.
- **tcpdump** — CLI-захоплення пакетів.
- **MQTT Explorer** — робота з MQTT.
- **QGroundControl** — GCS для дронів.

## Симуляція

- **MAVProxy** — MAVLink proxy.
- **Gazebo** — фізична симуляція.
- **jMAVSim** — PX4 SITL.

## Діаграми та документація

- **Mermaid** — діаграми в Markdown.
- **Excalidraw** — hand-drawn схеми.
- **MkDocs** — статичний сайт для документації.

## CI/CD та моніторинг

- **GitHub Actions** — CI/CD.
- **Prometheus + Grafana** — метрики.
- **Loki** — логи.
