# 01. Основи DefenseTech

## Що таке DefenseTech

DefenseTech — це технології, що використовуються для оборони, безпеки та військових застосувань. Для інженера-розробника це переважно:

- безпілотні системи (UAV / UGV / USV);
- наземні станції управління (GCS);
- системи зв’язку, навігації та телеметрії;
- комп’ютерний зір та AI для аналітики;
- надійні backend- та embedded-системи.

## Типи безпілотників

| Тип | Опис | Приклади |
|-----|------|----------|
| Fixed-wing | Літак із постійним крилом | Bayraktar TB2, Wing |
| Multirotor | Коптер з кількома роторами | DJI Mavic, FPV-дрони |
| VTOL | Комбінація вертикального зльоту і крила | Quantum-Systems Vector |
| UGV | Наземний робот | THeMIS, UGV platforms |
| USV | Надводний апарат | Maritime drones |

## Ключові компоненти UAV

- **Flight Controller** — польотний контролер (Pixhawk, Cube Orange).
- **ESC** — регулятори швидкості моторів.
- **GPS / GNSS** — навігація.
- **IMU** — інерціальний вимірювальний блок (акселерометр + гіроскоп).
- **Telemetry** — радіомодуль для зв’язку з наземною станцією.
- **Power Module** — контроль живлення.
- **Payload** — камера, скид, ретранслятор, датчики.

## Польотні контролери

- ArduPilot — відкрита, гнучка, широка підтримка платформ.
- PX4 — корпоративний стек, використовується в комерційних і оборонних дронах.
- Betaflight / INAV — для FPV та гоночних коптерів.

## Режими польоту

- MANUAL — повний ручний контроль.
- STABILIZE — стабілізація, але пілот керує.
- ALT_HOLD — утримання висоти.
- LOITER — утримання позиції.
- RTL (Return to Launch) — повернення на точку старту.
- AUTO — автоматичне виконання місії.
- GUIDED — керування через зовнішню команду (наприклад, REST API).

## Основи електронної боротьби (EW)

- **RF jamming** — глушіння каналу керування.
- **Spoofing** — підміна GPS-сигналу.
- **Detection** — виявлення дронів за радіосигналом.
- **Hardening** — захист від перерахованих загроз.

## Поглиблений огляд

### Основні концепції модуля 01

У цьому модулі ми розглянули ключові технології та підходи, які використовуються в сучасних DefenseTech системах. Кожна тема має практичне застосування: від embedded Linux і мережевих протоколів до AI і DevOps. Розуміння цих концепцій дозволяє будувати end-to-end рішення: дрони, наземні станції, backend, аналітика і розгортання.

### Практичне застосування

Теорія модуля має бути закріплена практикою. Рекомендується виконати лабораторну роботу, практичне завдання і мініпроєкт. Кожен наступний рівень складніший і ближчий до реального проєкту. Лабораторна дає базові навички, практика вчить самостійно вирішувати проблеми, мініпроєкт формує портфоліо.

### Масштабування

Коли рішення працює локально, важливо подумати про масштабування. Скільки дронів може обслуговувати система? Які протоколи використовувати для флоту? Як забезпечити відмовостійкість? Ці питання ми розглядаємо в наступних модулях, але вже на цьому етапі варто замислюватися про архітектуру.

### Інтеграція з іншими модулями

Модуль 01 не існує ізольовано. Його знання поєднуються з попередніми і наступними модулями. Наприклад, Linux і мережі використовуються в MAVLink, Python/C++ — для backend, ROS2 і CV — для AI, а DevOps — для розгортання. Курс побудований так, щоб кожен модуль доповнював загальну картину.

Для закріплення матеріалу використовуйте SITL, реальні embedded плати (Raspberry Pi, Jetson), симулятори, Docker, Kubernetes і хмарні сервіси. Чим більше практики, тим краще розуміння. Документація та спільноти допоможуть розібратися зі складними моментами.

### Часті питання

**Чи потрібен реальний дрон для навчання?** Ні, для більшості завдань достатньо SITL і симуляторів. Реальний дрон потрібен лише на пізніших етапах або для конкретних тестів.

**Чи можна вивчати модулі не по порядку?** Можна, але рекомендується послідовність, оскільки модулі будуються один на одному.

**Скільки часу потрібно на модуль?** Залежно від рівня і глибини — від 1 до 3 тижнів при рекомендованому режимі 30 годин на тиждень.

**Як перевірити, що я засвоїв модуль?** Виконайте чекліст модуля і мініпроєкт. Якщо можете пояснити матеріал іншій людині — ви його засвоїли.
