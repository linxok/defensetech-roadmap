# Готові комплекти дронів для початківця: аналіз для навчання ArduPilot/PX4 та модернізації

Аналіз доступних до продажу комплектів дронів вартістю до 200 $ для початківця, який планує вчитися прошивати flight controller, налаштовувати ArduPilot/PX4, підключати телеметрію та додавати companion board.

## TL;DR

**Для бюджету до 200 $ найкращий старт — комплект на базі Pixhawk 2.4.8 (CQ230 або аналогічний DIY-набір).** Він містить усе необхідне для першого польоту, сумісний із ArduPilot/PX4 і дозволяє поступово додавати телеметрію, GPS, companion computer та писати MAVLink-шлюз. Альтернатива — лише програмна симуляція (SITL) + QGroundControl, яка коштує 0 $ і є обов’язковою перед першим реальним польотом.

## Критерії вибору

- **Сумісність із ArduPilot/PX4.** Головна вимога, оскільки мета — навчитися прошивати та налаштовувати відкритий стек.
- **Наявність у комплекті:** FC, GPS, телеметрія, пульт, рама, мотори, ESC, пропелери, батарея, зарядка.
- **Розширюваність:** вільні UART, I2C, CAN, павербанк/5V для Raspberry Pi, можливість монтажу companion board.
- **Документація та спільнота.** Чим більше гайдів і відео — тим швидше пройти перші помилки.
- **Ремонтопридатність і ціна запчастин.** Для початківця crashes неминучі.
- **Ціна з урахуванням доставки в Україну.** Орієнтовно +20–40 % до ціни на AliExpress/китайському маркетплейсі.

## Детальний розбір варіантів

### 1. CQ230 Assembly Drone Development Kit — рекомендація для бюджету до 200 $

**Ціна:** $84.50–169.00 (залежно від продавця та комплектації; версія з Raspberry Pi 4B — дорожча).  
**Де шукати:** fpvzoom.com, rcdrone.top, gimbalmax.com, AliExpress.  
**Що входить:** Pixhawk 2.4.8, M8N GPS, CUAV WiFi Telemetry, 4×30A ESC, 4×2205 мотор, 4×5045 пропелер, рама CQ230 230 мм, 4S 2300mAh LiPo, балансний зарядник A400, пульт FS-i6, buzzer, optical flow module. У версії з RPi — Raspberry Pi 4B із Ubuntu 20.04, DroneKit, pymavlink, MAVROS, OpenCV, GStreamer.

**Чому підходить:**
- Ціна повного комплекту часто вкладається в 200 $.
- Вже сумісний із ArduPilot — можна одразу вчитися прошивати та налаштовувати.
- Містить optical flow (для утримання позиції без GPS у приміщенні).
- Версія з Raspberry Pi дозволяє почати робити MAVLink-шлюз і Python-скрипти з коробки.

**Обмеження:**
- Pixhawk 2.4.8 — застарілий клон; офіційна підтримка PX4 закінчується на v1.14, ArduPilot — на 4.4.x. Для навчання це не критично, але для серйозних проєктів краще Holybro Pixhawk 6C/6X.
- Невеликий час польоту (~7 хв), вага 612 г — після кількох вдосконалень доведеться міняти раму або батарею.
- Якість клонів може варіюватися; радимо купувати в продавців із відгуками.

### 2. DIY-набір на Pixhawk 2.4.8 — для тих, хто хоче зрозуміти кожен провід

**Орієнтовний бюджет:** $120–200.  
**Компоненти:**
- Pixhawk 2.4.8: ~$30–60 (AliExpress/Amazon).
- NEO-M8N GPS із компасом: ~$25–35.
- SiK telemetry radio 433/915 МГц: ~$30–60 (китайський клон) або Holybro SiK V3 ~$59.
- Пульт FS-i6 + приймач: ~$40–60.
- Рама 250–300 мм + мотори 2205/2212 + ESC 20–30A + пропелери: ~$40–80.
- Батарея 3S/4S LiPo + зарядний: ~$30–50.
- Power module, buzzer, кабелі: ~$10–20.

**Плюси:** повний контроль, дешеві запчастини, можна почати з більшої рами (легше літати на відкритому просторі).  
**Мінуси:** вимагає більше часу на збірку, пайку роз’ємів і налагодження; легко щось переплутати.

### 3. Holybro X500 v2 / Pixhawk 6C — для тих, хто готовий розтягнути бюджет або купувати поетапно

**Ціна:** повний комплект із Pixhawk 6C + GPS + SiK + рама + ESC + мотори — від ~$500.  
**Окремо FC:** Pixhawk 6C Mini — $122–193 (без урахування доставки та митниці).

**Чому варто знати:** це сучасний enterprise-рівень. Якщо мета — не просто погратися, а підготуватися до роботи в DefenseTech, варто знати саме такі контролери. Але в межах 200 $ можна дозволити собі тільки FC, а раму та мотори купити пізніше.

**Стратегія:** купити Holybro Pixhawk 6C Mini + GPS + телеметрію (~$200), тренуватися на SITL, а потім докуповувати раму/мотори. Або спочатку купити CQ230, навчитися, і на другому етапі переходити на Pixhawk 6C/6X.

### 4. SpeedyBee F405 V4 — не рекомендується для цієї мети

Хоча ціна приваблива ($66–76), він підтримує тільки Betaflight/INAV. ArduPilot/PX4 на F405 V4 не підтримуються. Для навчання прошивки ArduPilot/PX4 цей варіант не підходить.

## Рекомендований навчальний шлях модернізації

### Етап 0. SITL — безкоштовно, обов’язково перед польотом

- Встановити ArduPilot SITL або PX4 SITL.
- Підключити QGroundControl / Mission Planner.
- Пройти калібрування, арминг, режими польоту, взліт/посадка у симуляторі.
- **Мета:** зрозуміти, як прошивка, параметри та GCS взаємодіють, не ризикуючи залізом.

### Етап 1. Збірка та перша прошивка (~$85–170)

- Купити CQ230 або DIY-набір на Pixhawk 2.4.8.
- Зібрати раму, підключити FC, GPS, телеметрію, приймач пульта, ESC.
- Прошити ArduPilot Copter (рекомендовано для початківця) або PX4.
- Калібрувати: акселерометр, компас, ESC, пульт, рівень горизонту.
- Перший політ у режимі **Stabilize** або **AltHold** на відкритому просторі.

### Етап 2. Телеметрія та Ground Control Station

- Підключити SiK-radio 433 МГц (для України) або WiFi-телеметрію (ESP8266/ESP32).
- Навчитися зчитувати heartbeat, GPS, battery, ARM status через QGroundControl/Mission Planner.
- **Проєкт для портфоліо:** Python-скрипт, який підключається до MAVLink і виводить телеметрію в консоль (див. модуль 6 roadmap).

### Етап 3. Companion computer

- Додати Raspberry Pi Zero 2 W / Raspberry Pi 4 / Orange Pi (залежно від бюджету).
- Підключити до FC через UART/TELEM2.
- Встановити MAVProxy або mavlink-router.
- **Проєкт:** REST API на FastAPI, який отримує телеметрію від дрону (див. модуль 10 roadmap).

### Етап 4. Автономні місії та CV/AI

- Навчитися створювати waypoints, запускати AUTO-місії, RTL.
- Додати камеру + OpenCV / YOLO (для Raspberry Pi — легкі моделі, для Jetson — повноцінні).
- **Проєкт:** автоматична місія з виявленням об’єктів (див. модулі 11–13 roadmap).

## Ризики та обмеження бюджету до 200 $

- **Pixhawk 2.4.8 — клон.** Різниця в якості компонентів може бути значною; купуйте в перевірених продавців.
- **Не всі комплекти мають пульт.** Перевіряйте комплектацію перед покупкою.
- **Доставка в Україну.** Митниця + доставка можуть додати $20–50.
- **LiPo — небезпечні.** Потрібен балансний зарядний і контейнер для зберігання.
- **Перший польот — ризик крашу.** Мати запасні пропелери і раму з полікарбонату/ABS — дешевше ремонтувати.
- **Прошивка PX4 на старому 2.4.8.** Обмежена підтримка; для початківця рекомендується ArduPilot Copter.

## Висновок

Для бюджету до 200 $ і мети «вчитися прошивати та модернізувати» найкращий вибір — **готовий комплект CQ230 на базі Pixhawk 2.4.8** або **самостійна збірка на тому ж FC**. Вони дають повний цикл: збірка → прошивка ArduPilot → калібрування → політ → телеметрія → companion board → MAVLink/Python. Перед покупкою обов’язково пройти **SITL**; це заощадить і залізо, і нерви.
