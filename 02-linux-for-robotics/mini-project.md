# Мініпроєкт 02

## Linux для робототехніки

налаштувати systemd і udev для сервісу телеметрії

## Мета

Побудувати закінчений компонент, який можна показати в портфоліо.

## Вимоги

1. Використовуйте технології з цього модуля.
2. Створіть окремий репозиторій або папку.
3. Додайте README з інструкцією запуску.
4. Покрийте код базовими тестами або чеклістом.
5. Додайте демонстрацію: скріншот, відео або лог.

## Технічний стек

- Технології з модуля Linux для робототехніки.
- Docker (опціонально).
- CI/CD (опціонально).

## Етапи

1. Проєктування архітектури.
2. Реалізація core функціоналу.
3. Тестування.
4. Документація.
5. Публікація.

## Критерії завершення

- Проєкт запускається.
- Є документація.
