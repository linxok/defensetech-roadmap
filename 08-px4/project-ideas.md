## Як вибрати проєкт

Обирайте проєкт, який:
- вирішує реальну задачу;
- використовує технології поточного модуля;
- можна завершити за 1–2 тижні;
- підходить для портфоліо.

## Що включити

- README з інструкцією.
- Код і тести.
- Docker або інструкцію з налаштування.
- Демо: скріншот, відео або лог.
