# 07. ArduPilot

## ArduPilot

ArduPilot — відкритий автопілот, що підтримує multicopters, planes, rovers, boats, submarines. Має потужну SITL (Software In The Loop) для тестування без реального дрона.

## Теми модуля

- **SITL** — запуск симуляції на ПК.
- **QGroundControl** — GCS для конфігурації та моніторингу.
- **Режими польоту**: RTL, AUTO, GUIDED, LOITER.
- **Lua scripting** — користувацькі скрипти на автопілоті.

## Lua scripting

ArduPilot підтримує Lua-скрипти для користувацької логіки на FC.

```lua
function update()
    gcs:send_text(0, "Hello from Lua")
    return update, 1000
end
return update, 1000
```

## GUIDED mode з Python

```python
from pymavlink import mavutil

conn = mavutil.mavlink_connection('tcp:127.0.0.1:5762')
conn.wait_heartbeat()
conn.set_mode('GUIDED')
conn.arducopter_arm()
conn.mav.command_long_send(
    conn.target_system, conn.target_component,
    mavutil.mavlink.MAV_CMD_NAV_TAKEOFF, 0,
    0, 0, 0, 0, 0, 0, 10
)
```
