# Практичне завдання 07

## Завдання

ArduPilot: керувати симульованим UAV через API

## Опис

Повторіть лабораторну роботу без підказок, додайте логування, обробку помилок та метрики.

## Що зробити

1. Виконайте всі кроки лабораторної самостійно.
2. Додайте структуроване логування.
3. Додайте обробку типових помилок (timeout, disconnect, invalid data).
4. Запишіть метрики: час виконання, CPU, memory, throughput.
5. Покрийте код базовими тестами, де це можливо.

## Результат

Покращений приклад у власному репозиторії з README.
