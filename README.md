# DefenseTech Roadmap

> Open-source learning path from Senior Full-Stack Developer to DefenseTech / Drone / Robotics Software Engineer.

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

## About this course

This repository is a structured, hands-on course that converts a single roadmap into a public, portfolio-grade body of knowledge. It covers the full stack needed for modern defense and aerospace software engineering: embedded Linux, networking, Python/C++ backends, MAVLink, ArduPilot, PX4, ROS2, computer vision, AI, ground control stations, and DevOps.

- **Navigation and README are in English** to keep the project accessible on GitHub.
- **Main content is in Ukrainian** for local learners and defense-tech communities.

## Target roles

- AI Software Engineer
- Backend Engineer (DefenseTech)
- Robotics Software Engineer
- Drone Software Engineer
- Ground Control Software Engineer
- Mission Planning Engineer

## How to use this course

1. Start with `00-introduction/`.
2. Follow modules in order; each module contains theory, lab, practice, code examples, homework, and a mini-project.
3. Use the `checklist.md` at the end of each module to verify your knowledge.
4. Build the portfolio projects in `16-projects/`.
5. Prepare for interviews using `17-interview/`.

## Weekly routine recommendation

- 10 hours theory
- 10 hours coding
- 5 hours documentation
- 5 hours portfolio
